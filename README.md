# 🗒️ NoteVault

A production-ready Notes Keeper web app built with React, Supabase, and Tailwind CSS.

## 🌍 Live Demo

👉 [https://notes-app-zowq.vercel.app](https://notes-app-zowq.vercel.app)

## ✨ Features

- 🔐 Email & Password Authentication
- 📝 Create, Edit, Delete Notes
- 🔍 Instant Search & Filter
- 📱 Mobile Responsive
- ⚡ Optimistic UI Updates
- 🔔 Toast Notifications

## 🧰 Tech Stack

- **Frontend** - React + Vite
- **Styling** - Tailwind CSS
- **Backend** - Supabase (Auth + Database)
- **Deployment** - Vercel

## 🚀 Getting Started

1. Clone the repo
```bash
   git clone https://github.com/Gagana-bm/notes-app.git
```

2. Install dependencies
```bash
   npm install
```

3. Add your `.env` file
